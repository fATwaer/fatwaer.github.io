elasticsearch 文档创建和删除 创建和更新文档 官方提供的接口: PUT /<target>/_doc/<_id> POST /<target>/_doc/ PUT /<target>/_create/<_id> POST /<target>/_create/<_id> 下面的例子利用PUT方法创建一个 id 为1的文档： PUT /index-001/_doc/1 { "title": "Elasticsearch: The Definitive Guide", "authors": [ "clinton gormley", "zachary tong" ], "summary": "A distibuted real-time search and analytics engine", "publish_date": "2015-02-07", "num_reviews": 20, "publisher": "oreilly" } 响应结构为： { "_index" : "index-001", "_type" : "_doc", "_id" : "1", "_version" : 1, "result" : "created", "_shards" : { "total" : 2, "successful" : 1, "failed" : 0 }, "_seq_no" : 18, "_primary_term" : 1 } 其中，version字段为1，并且result的值为created。
